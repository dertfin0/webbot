# WebBot
WebBot - простое решение для создания бота по управлению чем-либо

```commandline
pip install webbot_lib
```

**Как это работает?**
Проект состоит из трёх модулей: `api`, `cli-client` и `library`. 
1. Пользователь поднимает на своём сервере `api` - он обеспечивает связь между пользователем и ботом. 
2. Пользователь создаёт бота при помощи библиотеки из модуля `library` - самый простой бот занимает всего пару строк кода, а
названия большинства команд совпадают с `pytelegrambotapi` - самой популярной библиотекой для создания подобных ботов через Telegram
3. После настройки сервера и бота, пользователь может воспользоваться `cli-client`, чтобы взаимодействовать с ботом
через простой легковесный интерфейс

### Как поднять сервер? (api)
Для того чтобы Вы могли взаимодействовать с ботом из любой точки мира, Вам необходимо арендовать
или сделать свой простой VPS-сервер - он является связующим звеном между пользователем и ботом.

#### Минимальные требования
- Ubuntu/Debian 2020 года и позднее (можно использовать Windows, но он отсутствует в руководстве)
- 1 ядро CPU
- 512 МБ свободного ОЗУ
- 10 Гб общего места на диске (из них необходимо ~500 Мб свободных)

#### Установка и запуск API
1. Скачайте `git` и `python3`:
    ```bash
    sudo apt update
    sudo apt install git python3-full -y
    ```
2. Скачайте файлы WebBot
    ```bash
    git clone https://github.com/dertfin0/webbot
    cd webbot/api
    ```
3. Создайте venv и установите всё необходимое для работы:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    ```
4. Создайте файл с настройками и укажите пользовательский пароль, а также пароль для бота
    ```bash
    echo "USER_PASSOWRD=ВАШ_ПАРОЛЬ" >> .env
    echo "BOT_TOKEN=ПАРОЛЬ_ДЛЯ_БОТА" >> .env
    ```
5. Запустите сервер:
    ```bash
    nohup python3 main.py & 
    ```

### Создание бота
Чтобы создать своего бота, нужно установить библиотеку `webbot`:
```commandline
pip install webbot
```

Давайте создадим простого бота, который будет повторять сообщения:
```python
from webbotz import WebBot

# Создаём бота. Тут нужно указать адрес вашего сервера и пароль для бота, который был указан в .env
bot = WebBot("http://адрес_вашего_сервера:8000", "пароль_для_бота")

# Создаём обработчик сообщений
@bot.message_handler()
def echo(message):
    bot.send_message(message.text) # Отправляем ответ - сообщение с тем же текстом, который отправил пользователь

bot.infinity_polling() # Запускаем бота
```

Можем сделать обработку нескольких команд:
```python
import os 

bot = WebBot("http://адрес_вашего_сервера:8000", "пароль_для_бота")

@bot.message_handler()
def start(message):
    if message.text != "/start":
        return
    bot.send_message("Hello world!")

# Можно сделать удаленное выключение своего компьютера
@bot.message_handler()
def shutdown(message):
    if message.text != "/shutdown":
        return
    os.system("shutdown -s -t 0")

bot.infinity_polling()
```

#### Создание команд в несколько этапов
```python
bot = WebBot("http://адрес_вашего_сервера:8000", "пароль_для_бота")

@bot.message_handler()
def hello(message):
    if message.text != "/hello":
        return
    bot.send_message("Как тебя зовут?")
    bot.register_next_step_handler(handle_hello) 
    # Следующее сообщение получит только handle_hello

def handle_hello(message):
    name = message.text
    bot.send_message("Привет, " + name + "!")
```

#### Продвинутые настройки
При создании бота можно указать также некоторые необязательные параметры:

```python
import logging

bot = WebBot("http://адрес_вашего_сервера:8000", "пароль_для_бота",
             cooldown=0.5,  # Проверять сообщения каждые 0.5 секунд (по умолчанию - 1)
             timeout=3,  # Выдавать ошибку таймаута через 3 секунды (по умолчанию - 5)
             logger=logging.getLogger() # Указание своего логгера - можно изменить формат сообщений в консоли, выводить ошибки в файл итд
)
```

### Взаимодействие с ботом (клиент)
Чтобы общаться с ботом, нужно установить `cli-client`. 

Для начала, создайте новый файл, например `config.txt`, и настройте его:
```
SERVER_ADDRESS=http://адрес_вашего_сервера:8000
PASSWORD=пароль_для_пользователя
```

Если у вас Windows *(x64)* или Linux *(x86_64)*, то зайдите на [страницу с последним релизом](https://github.com/dertfin0/webbot/releases/latest) и установите клиент.

Скачанный файл поместите **в одной папке** с `config.txt` и запустите клиент через командную строку:

Windows:
```
webbot config.txt
```

Linux:
```
chmod +x webbot
./webbot config.txt
```

#### Запуск на других платформах
Вы можете собрать клиент под себя:
1. Установите Python на ваш ПК
2. Установите исходники webbot со страницы релиза, они помечены как `Source code`, и распакуйте их
3. Перейдите в папку `cli-client`:
    ```commandline
    cd cli-client
    ```
4. Создайте venv и установите всё необходимое:
    Windows:
    ```commandline
    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements.txt
    ```
    Linux:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    ```
5. Запустите скрипт для сборки своего клиента:
    Windows:
    ```commandline
    build
    ```
    Linux:
    ```bash
    chmod +x build.sh
    ./build.sh
    ```
6. После сборки вы можете найти рабочий клиент в папке `dist`, откуда можете перенести его в удобное место и
запустить по описанной выше инструкции
